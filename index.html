Frequently Asked Questions

1. The difference between var, let, and const

var

  1. Function Scoped
  2. Initialized as undefined at the top of the Scope
  3. Can be redeclared
  4. Can be reassigned

let

  1. Block Scoped
  2. Not initialized before declaring
  3. Can not be redeclared
  4. Can be reassigned

const

  1. Block Scoped
  2. Not initialized before declaring
  3. Can not be redeclared
  4. Can not be reassigned
2. The difference between map(), forEach(), and filter()

map()

Create a new array with changed elements

transform array elements

must return

forEach()

Do not return a new array

perform operation on individual elements

return nothing

filter()

Return a new array

filter elements based on logic

return boolean

3. Explain arrow functions and how they are different from regular functions

The syntax of Arrow function is cleaner and shorter than the regular function. It doest not require function keyword to declare. Also if the function has only one statement the {} is not required. Very useful during callbacks.

4. How JavaScript Promises work?

A promise is Javascript is an asynchronus operation which has three states:
1. Pending
2. Resolved
3. Rejected

5. How closures work in JavaScript?

Closures are functions that store references to variables in functions outside them and can access them even after the function has finished.
